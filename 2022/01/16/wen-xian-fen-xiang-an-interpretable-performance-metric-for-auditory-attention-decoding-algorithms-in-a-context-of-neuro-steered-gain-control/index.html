文献分享||An Interpretable Performance Metric for Auditory Attention Decoding Algorithms in a Context of Neuro-Steered Gain Control

发表于2022-01-16|更新于2022-01-16|Biomedical Engineering
|字数总计:1.9k|阅读时长:5分钟|阅读量:

文献信息

文章概览

  在多说话者的场景中，即使听力损失患者已经佩戴助听器，助听器也可能会在增强竞争声音的同时，错误地将被患者更加关注的声音视为噪声，且目前有文献已经证明从EEG数据中解码患者更加关注的声音的可行性，即听觉注意检测，但是对于目前所提出的诸多AAD算法性能没有统一的评估标准。此外由于AAD算法的精确度以注意力被正确解码的决策窗口的百分比来衡量，取决于决策窗口的长度，决策窗口的长度定义了有多少脑电图数据可用于做出决策。由于脑电信号对语音信号的神经响应的信噪比(SNR)较低，因此准确率随判决窗长度的增加而提高。然而，较长的判决窗口意味着该算法还需要更多的时间来例如对注意力的切换做出反应，这导致了二者之间的权衡。本文提出了一种可解释的性能度量来评估AAD算法，该算法基于自适应增益控制系统，结合了神经解码器的准确率和判决窗口长度，由AAD判决指导，这样的系统可以被建模为马尔可夫链系统，由此可以计算最小预期切换持续时间(MESD)，并将其解释为用户听觉注意力切换所需的最短预期时间。

亮点与创新点

  解码准确率和判决窗口长度是AAD较为重要的指标，本文首次提出最小预期切换持续时间(MESD)来评估AAD检测算法，从而解决AAD精度和决策时间之间的取舍问题。

图文解析

1）AAD算法性能评估问题的提出

图1 AAD算法存在的问题

  如图1所示，在前60s，说话人S1的声音被更加关注，而在60s之后，被试的听觉注意切换到说话人S2的声音，分析得出：在具有判决窗口长度τ和AAD精度的算法过程中存在两个基本问题，当增益水平的数量减少时（从A状态切换到B状态的过渡过程中，变化水平数量减少），即使增益切换执行得更快，但总体增益过程不太稳定，说明被试在听觉注意力切换过程的听觉感受并不舒适；当减小判决窗口长度时，虽然增益切换更快但是解码准确率显著下降， 被试在听觉注意力切换过程的听觉感受不舒适的同时，所解码的听觉信息也不够准确。

2）马尔科夫链模型的建立

图2 自适应增益控制系统建模为马尔可夫链

  听觉注意过程中的自适应增益控制系统可以直接转化为使用马尔可夫链的数学模型，其中马尔可夫链包含N个状态，每个状态对应于更被关注的说话人和竞争说话人以及干扰说话人在内的背景噪声的相对增益水平，在此为了不失一般性，本文考虑的都是无噪音的两个说话人的场景。我对该模型的理解如下：被试的听觉注意从A切换到B的过程中，假设x=1对应于被更加关注说话人的目标增益水平，则目标方向上的转移概率p∈[0，1]等于AAD精度，类似地，q=1−p对应于错误决策的概率，可以在下文中，我们假设p>0.5，即，评估的AAD算法的性能至少好于机会水平，每一步都需要τ秒(决策窗口长度)，且在做出新的决策之前，需要缓冲τ秒的EEG数据和音频数据，在τ秒的连续窗口上应用AAD算法，从而产生如图1所示的增益过程，即对应于马尔可夫链的随机行走过程。

3）MESD计算步骤

图3

  对N、τ和p进行优化，得到最优马尔可夫链，产生最佳工作点(τOPT，POPT)的P(τ)-性能曲线上的最小ESD称为最小预期切换持续时间(MESD)， MESD 分以下四个步骤计算：

  ①p(tau) 性能曲线的构建，方法是通过评估的（在真实 EEG 和音频数据上）（tau_i，p_i）点（决策窗口长度精度）进行插值

  ②优化 p(tau) 性能曲线上每个采样 tau 的状态数 N ，得到最优的马尔可夫链

  ③计算每个采样 tau 的预期开关持续时间 ESD(p(tau),tau,N_tau) 和相应的最佳状态数 N_tau

  ④所有评估过的 ESD 中的最小值即为MESD

4）应用MESD对AAD算法进行性能评估

  通过对数据集包括每个受试者(16名正常听力受试者)72分钟的记录脑电和音频数据进行听觉注意解码，这些受试者被指示在竞争的两个说话者的情况下选择听特定的语音刺激，并进行对应的EEG数据预处理工作： 将64通道的EEG数据在1 Hz到9 Hz之间进行带通滤波，然后下采样到20 Hz，通过子带滤波之后，使用指数为0.6的幂律运算来计算语音包络，然后进行类似的带通滤波和下采样；并且从麦克风录音中提取语音的实际语音包络，计算预测语音包络和呈现给对象的两个信号的包络之间的皮尔逊相关系数，具有最高相关性的语音流就被标识为更加关注的声音。

图4 平均自相关矩阵的AAD算法性能和解码器平均的AAD算法性能比较

  如图4所示，比较平均自相关矩阵和平均解码器两种不同的AAD方法，可以得出两种不同的结论：当观察决策时间窗口τ>30s时，两种不同的AAD方法性能基本一致；然而，在τ<30s时，尽管总共使用了等量的训练数据，但平均自相关矩阵的准确率优于平均解码器，MESD将较小的判决窗长度作为性能曲线的相关部分，因此借助MESD对于不同的AAD算法可以找到一个最优的决策时间窗口，最大限度保证解码准确率。

启示与思考

  1）准确率和决策时间窗口是AAD的主要衡量指标，使用MESD来综合评价两个指标可以保证最优的解码算法性能，即高效和准确性。

  2）将MESD加入到AAD过程中，本文作者提供了MESD的MATLAB工具箱以及Python源码，可以修改设定默认超参数：P_0 = 0.8（置信水平）、c = 0.65（下限置信区间）、N_min = 5（最小状态数）和 K = 1000（在性能曲线上评估的样本数）调，来调整自己的AAD算法。
