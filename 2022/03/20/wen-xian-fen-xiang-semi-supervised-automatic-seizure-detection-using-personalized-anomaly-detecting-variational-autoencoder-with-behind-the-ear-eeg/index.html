文献分享||Semi-supervised automatic seizure detection using personalized anomaly detecting variational autoencoder with behind-the-ear EEG

发表于2022-03-20|更新于2022-03-20|Neuroscience
|字数总计:995|阅读时长:3分钟|阅读量:

文献信息

文章概览

  我们收集了该医院16例癫痫患者的耳后脑电图信号，并利用它们来开发和评估癫痫发作检测算法。我们对变分自编码器网络进行了改进，以学习正常脑电图信号的潜在表示，并利用训练后的网络测量脑电图信号的异常来进行癫痫检测。为了个性化该算法，我们还提出了一种方法，通过比较潜在空间中的表示形式来校准每个患者的异常评分。我们提出的算法在没有个人校准的情况下，灵敏度为90.4%，误报率为每小时0.83。另一方面，单类支持向量机的灵敏度仅为84.6%，误报率为每小时2.17。此外，我们提出的个人校准模型在检测52个发作事件中的49个时，达到了94.2%的灵敏度，误报率为0.29。

亮点与创新点

  ① 通过异常检测变异自动编码器进行半监督方法，通过耳后脑电图检测癫痫发作

  ② 利用来自16例癫痫患者的500多小时耳后脑电图数据，开发并进行了临床评估

  ③ 开发的算法实现了灵敏度为94.2%的检测性能，平均每小时0.29的误报率，用于交叉验证

图文解析

  耳后脑电图的电极位置。LT：左侧颞电极；RT：右侧颞电极；LC：左中心电极；RC：右中心电极。交叉头通道是通过向两侧连接电极来构建的。

  受试者工作特征曲线下患者级面积的比较。SVM：支持向量机；AnoVAE：异常检测变分自动编码器。星号（∗）表示来自所有其他算法的显著性。采用Welch的t检验来确认它们之间的显著性。

  正常发作期和发作期脑电图对数功率的比较。大部分频带在正常发作期和发作期之间有显著性差异。在发作样本中，Delta, beta, alpha, beta, low-gamma, 和high-gamma波段显示出显著的波段功率增加。另一方面，纹带在发作样本中没有明显的变化。采用Welch的t检验来确认它们之间的显著性。

  通过t-SNE可视化AnoVAE的潜在表示。在16维中的潜在表示被压缩为三维，并使用t-SNE(t-随机邻域嵌入)进行可视化。黄色样本表示发作期对应的脑电图序列，蓝色样本表示非发作期（正常）样本。

  可视化的耳后脑电图信号的频谱模式的误差分析。这些图显示了单个患者在发作间隔周围的功率谱变化。红色虚线表示具有侵入性注释的发作性开始和偏移量。在图(a)的情况下，几乎在发作后就可以观察到不规则的模式，如功率的瞬间上升。（黑色箭头）在图(b)中，一些不规则的功率增加刚好在发病前很接近。（黑色箭头）另一方面，在(c)图中，在整个发作间隔期间未观察到任何异常模式。

启示与思考

  本文提出了一种新型的癫痫发作检测算法，该算法通过对异常检测变分自动编码器的半监督学习和通过比较潜在表征的异常评分个性化方法，对耳后脑电信号进行了检测。我们的方法通过高灵敏度和更低的误报率实现了改进的癫痫发作检测。本文提供了一个耳脑电可能的应用场景。
