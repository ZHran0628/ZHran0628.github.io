文献分享||Decoding imagined speech from EEG signals using hybrid-scale spatial-temporal dilated convolution network

发表于2022-04-03|更新于2022-04-03|neuroscience
|字数总计:2.3k|阅读时长:6分钟|阅读量:

文献信息

文章概览

  直接解码想象发声的EEG信号在脑机接口应用中引起了人们的极大兴趣，目前已有多种方法应用于想象发声脑电信号的解码，但如何在脑电信号中构建时空相关性并捕捉语音的上下文线索以更好地解码想象语音仍是需要考虑的问题。在本研究中提出混合尺度时空膨胀卷积网络(HS-STDCN)将时间和空间信息的特征学习集成到一个统一的端到端模型中，捕获多个层次的时间信息来刻画脑电信号序列的时间相关性，然后利用深度空间卷积层来构建脑电信号的固有空间关系，该空间关系可以产生输入脑电数据的时空表示。在时空表示的基础上，进一步利用膨胀卷积层学习特征，最终实现分类。为了对所提出的方法进行评估，本文也在收集的数据集上将HS-STDCN与其他已有方法进行比较。

亮点与创新点

  1)本研究中提出的HS-STDCN 的端到端网络架构，利用时空判别信息来解码想象发声脑电信号，为了减轻潜在的时间相关伪影的影响并使实验结果更加可靠，以随机顺序跨不同块收集想象发，对8个想象发声词语的平均分类正确率为54.31%，解码性能显著优于其他方法(p<0.05)

  2)考虑到视觉呈现可以更好地诱导信息的学习、回忆和检索，本研究中使用图片作为语音内容的提示，并为每个词语类别选择了十张图片以避免单张图片信息的影响

  3)本研究还基于 HS-STDCN 模型对词语的语义差异进行了可视化分析，研究语义对想象发声脑电信号解码的影响，同时研究与EEG信号解码相关的脑区，使用较少的电极来比较解码性能

图文解析

图1 完整实验流程

  图1显示了一个详细的实验过程：在实验开始之前被试有 3-5s的准备时间，同时让被试调整到舒适的坐姿；随后要求被试将注意力集中在屏幕中央显示的注视十字；然后从每个类别对应的10张图片中随机选择1张图片并显示 1s，在 图片显示时间内，被试首先判断图片，然后在脑海中想象说出相应的中文词语，实际并不发声或也不做出发音肌肉运动，当8个词语中的每一个都执行一次后，允许受试者休息 5s，将每次采集到的1s的脑电数据输入至混合尺度时空膨胀卷积网络进行解码。

图2 HS-STDCN 框架

  图2显示了所提出的 HS-STDCN 模型的框架从多通道 EEG 信号中捕获远程时间和空间信息，该网络首先使用混合尺度时间卷积和空间卷积来学习时空表示；随后具有增量感受野的扩张卷积被用于进一步利用高级差异信息；最后使用全连接层进行分类，对于采集到的所有脑电信号以 8:2 的比例分为训练集和测试集，使得训练样本和测试样本来自不同的块，进一步减轻了时间相关性对分类的影响，使实验结果更加可信。

图3 HS-STDCN 模型与其他模型性能对比

  为了评估提出的 HS-STDCN 模型，本研究对以平均准确度（ACC）和标准差（STD）作为模型性能的评价标准，此外为了验证 HS-STDCN 的分类优越性，还将HS-STDCN 模型的平均分类准确率与先前研究中取得显着性能的深度学习方法进行了相同的实验，包括 CSP 和 SVM 、DWT 和 MLP ，统计特征和 ELM，切线向量和 RVM 以及EEGNet ，所有模型和方法的分类结果如图3所示。

图4不同词语之间的语义差异可视化

  为了进一步了解词语义对想象发声脑电信号解码性能的影响，本研究还探讨了不同类型词语的语义关系，通过MDS运算，输出多维向量被映射成二维散点图以可视化不同单词之间的距离，然后根据 MDS 运算的结果计算出每种单词的质心，结果如图4所示，并获得3个观察结果：
  1)所有受试者的水果词和动物词之间的语义差距都很明显，这与人类的正常语义感知一致，这一观察表明 HS-STDCN 可以从想象的语音脑电信号中学习丰富的语义信息。

  2)四个动物词之间的语义差异大于四个水果词的语义差异。考虑到所选择的四种动物在日常生活中并不常见，它们之间的联系并不比四种水果更紧密，这也反映了想象言语活动中的潜在语义感知。

  3)受试者在意象词中表现出语义感知差异：对于水果词和动物词，它们之间的语义差异因每个主题而异，例如 S9 的四个水果词之间的距离比其他科目更近，S3 的四个动物词之间的距离比其他种类更紧凑，这可能是由于语义处理的个体差异。

图5基于 HS-STDCN 的不同想象发声词语的脑电电极活动图

  此外，为了进一步研究在HS-STDCN中获得的判别信息来自不同的意象词，每个意象词对应的电极活动图如图5所示，得到以下观察结果：

  1)对于四个水果词，激活的大脑区域大致相同。图5显示顶叶的贡献大于其他区域。对于“西瓜”，左后颞叶与顶叶具有相同的贡献。

  2)对于四个动物词，激活区域表现出一些差异。图5 显示，对于“狒狒”和“斑马”，顶叶和左后颞叶都做出了重大贡献。对于“狮子”，最活跃的区域集中在左后颞叶。对于“犀牛”，顶叶和左下额叶的贡献最大。

图6选定电极的位置

  对于脑机接口系统，使用更少的电极可以大大提高应用的可行性和便利性。在本节中，本文还研究所提出的 HS-STDCN 模型的解码性能如何随着电极的减少而变化：基于图6所示的结果，研究过程中从顶叶、左后颞叶和左下额叶等更多贡献区域中选择了不同数量的电极来进行想象的语音识别。这些选定的电极被称为此外，由于水果词和动物词之间的差异，我们从右颞叶和右枕叶中选择了几个电极:

  1)使用较少电极获得的结果与使用所有 64 个电极获得的结果相当，与全部 64 个电极相比，基于 32 个电极的性能仅下降 4%；同样使用 16 个电极也实现了显着的解码性能，比使用 32 个电极低 2%。结果验证了语言功能区域被激活并且对想象的语音解码任务的贡献更大

  2)右脑在想象的语音识别中也发挥了重要作用：基于从右半球选择的六个电极的性能，比10个电极的性能提高了 2%，这一结果表明，右半球也参与了语言相关活动的处理

启示与思考

  1)实验范式的改进：在我未来的实验设计中，可以分别利用视觉刺激和听觉刺激，让被试进行想象发声，最后对比两种不同刺激范式对脑电信号解码准确率的影响

  2)本研究对词语语义的分析，是值得进行学习的：不同词语对脑区的激活响应不一样，在目前的研究结果来看：想要实现对连续语句的解码是存在难度的，不过可以从单个数字或者字母的研究逐渐拓展到对词语、语句的研究，有利于我们理解想象发声的实现机制。
